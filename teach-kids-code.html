Teach Your Kid To Code

I believe in empowering students of all ages and backgrounds to become innovative problem solvers and technology leaders. Every student can engage in meaningful, hands-on learning experiences that spark curiosity and inspire a lifelong passion for technology.

This website is offered only to individuals. It is illegal to use this collection to source content for your paid course. For-profit educators interested in using this page to help with curricula must write for permission first.
